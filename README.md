# NapCat OpenClaw Extension

NapCat QQ 的 OpenClaw 扩展插件，提供 QQ 消息收发、流式传输、用户信息解析等功能。

## 功能特性

- QQ 消息收发支持
- 流式传输（typing indicator）
- 消息解析与格式化
- 用户信息缓存
- SQLite 数据库支持

## 安装

```bash
# 克隆仓库
git clone https://github.com/Hesitate-P/napcat-openclaw.git

# 安装依赖
npm install

# 构建
npm run build
```

## 配置

在 OpenClaw 配置文件中添加插件路径即可。

## 开发

```bash
# 开发模式
npm run dev

# 测试
npm test
```
